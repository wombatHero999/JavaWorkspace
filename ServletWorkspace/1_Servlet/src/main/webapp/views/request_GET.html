Get 방식으로 요청 후 응답페이지 받아보기

특징 1. GET방식으로 요청하는건 URL의 Header영역에 데이터들을 포함시켜서 요청하는 것을 의미한다.
=> 사용자가 입력한 값들이 url에 쿼리스트링으로 노출된다
=> 데이터 보안이 불가능
=> 즉, 로그인 , 회원가입시 부적절한 방법
특징 2. Header영역은 전송길이 제한
=> 방대한 데이터를 담은 경우, 초과된 데이터는 절단
=> 즉, 게시판 등록, 수정과 같은 방식에서는 부적절한 방법
